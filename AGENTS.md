# 프로젝트 개요

- 수학학원에서 약 18-27인치의 터치스크린을 이용해 시험을 응시하는 모의고사 웹앱 개발
- 모의고사는 학생들이 지면에 문제를 풀고 답안을 앱으로 마킹하여 채점과 결과 확인, 이후 복습까지 온라인으로 진행되는 방식
- [현재 사용중인 데모](https://edu.basemath.co.kr/demo/exam)

## 기능 명세

모의고사 앱은 크게 세 단계로 구성됩니다.

- 튜토리얼: 시험 응시에 대한 정보와 OMR 카드 작성 방법을 안내합니다.
- 답안지 마킹: 시험 답안을 OMR 카드에 입력하고, 남은 시간 등을 표시합니다.
- 채점 및 결과 확인: 답안을 제출하고 결과를 확인합니다.

## 기술 스택

- 프론트엔드는 React + Vite 템플릿 기준입니다.
- 패키지 관리는 pnpm을 사용합니다.
- 스타일링은 Tailwind CSS v4를 사용합니다.
- 데이터 관리는 TanStack Query를 사용합니다.
- 그 외 스택은 자유롭게 선택할 수 있습니다.
